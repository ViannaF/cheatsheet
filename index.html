CSS Selector Reference

Simple Selectors

Selector Description Example Example Description
element (Type) Selects HTML elements by element name p Selects all <p> elements
#id Selects an element by its unique id attribute #cheatsheat Selects the element with id=cheatsheat
.class Selects elements by class atrribute .container Selects all elements with class=container
* (Universal) affects every HTML element on the page
element.class (Chaining Selectors) Selects only elements that fulfill all criteria specified p.simple Selects <p> elements with class=simple
element,element (Grouping Selector) Selects elements to which the same ruleset will be applied h1, h2, p Selects all <h1>, <h2> and <p> elements

Combinator Selectors

Selector Description Example Example Description
element element (Descendant Selector) Selects elements that are descendents of specified element div p Selects all <p> elements that are inside <div> elements
element>element (Child Selector) Selects elements that are children of specified element div > p Selects all <p> elements that are children of a <div> element
element+element (Adjacent Sibling Selector) Selects an element immediately following another element. Both elements must share the same parent element. div + p Selects the first <p> element that directly follows the <div> element.
element1~element2 (General Sibling Selector) Selects all elements that are next siblings of an element div ~ p Selects all <p> elements that follow the <div> element

Pseudo Classes

Selector Description Example Example Description
:active Selects an active element a:active Selects the active link
:checked Selects every checked <input> element input:checked Selects every checked <input> element
:disabled Selects every disabled <input> element input:disabled Selects every disabled <input> element
:empty Selects the specified elements with no children p:empty Selects every <p> element with no children
:enabled Selects every enabled <input> element input:enabled Selects every enabled <input> element
:first-child Selects the specified element that is the first-child of another element p:first-child Selects every <p> element that is the first child of its parent
:first-of-type Selects the first element of its type among a group of sibling elements p:first-of-type Selects every <p> element that is the first <p> element of its parent
:focus Selects an element that has focus input:focus Selects the <input> element that has focus
:hover Selects element when cursor (mouse pointer) hovers over it a:hover Selects a link on mouse over
:in-range Selects <input> elements with a value in the specified range input:in-range Selects <input> elements with a value in the specified range
:invalid Selects any <form>, <fieldset>, <input> or other <form> element whose content fails to validate input:invalid Selects all <input> elements with an invalid value
:lang(language) Selects elements in certain languages p:lang(it) Selects every <p> element that has a language starting with "it"
:last-child Selects the specified element that is the last-child of another element p:last-child Selects every <p> element that is the last child of its parent
:last-of-type Selects the last element of its type among a group of sibling elements p:first-of-type Selects every <p> element that is the last child of its parent
:link Selects an element that has not been visited yet a:link Selects all unvisited links
:not() Selects all elements that are not the specified element :not(p) Selects every element that is not a <p> element
:nth-child(n) Selects elements based on the order they appear in the list of children of their parent p:nth-child(3) Selects every <p> element that is the third child of its parent
:nth-last-child(n) Selects elements based on the order they appear in the list of children of their parents counting from the last child p:nth-last-child(3) Selects every <p> element that is the third child of its parent counting from the end
:nth-last-of-type(n) Selects elements based on their position in a list of sibling elements of the same type, counting from the last child p:nth-last-of-type(3) Selects every <p> element that is the third <p> element of its parent, counting from the end
:nth-of-type(n) Selects elements based on their position in a list of sibling elements of the same type p:nth-last-of-type(3) Selects every <p> element that is the third <p> element of its parent
:only-of-type Selects elements that have no other siblings of the same type p:only-of-type Selects every <p> element that is the only <p> element of its parent
:only-child Selects an element that is the only child of its parent element p:only-child Selects every <p> element that is an only child of its parent element
:optional Selects any <input>, <select> or <textarea> element that does not have the "required" attribute set input:optional Selects <input> elements that do not have the required attribute set
:out-of-range Selects an <input> element whose current value is outside the range set by the min and max attributes input:out-of-range Selects <input> elements whose current value is outside the specified range
:read-only Selects an element that the user cannot edit input:read-only Selects <input> elements with the readonly attribute specified
:read-write Selects an element that the user can edit input:read-write Selects <input> elements that do not have the readonly attribute specified
:required Selects elements that have the required attribute set input:required Selects <input> elements that have the required attribute specified
:root Selects the root element of the document. In HTML this is identical to the html selector but has higher specificity
:target Selects a target that has an id matching the URL's fragment #info:target Selects the target (that has a matching id attribute) of the #info element when active
:valid Selects any input or form elements whose contents validate successfully input:valid Selects <input> elements with a valid value
:visited Selects links that have been visited by the user a:visited Selects visited links

Pseudo-elements

Selector Description Example Example Description
::after Used to insert content after the content of an element p::after Insert content after the content of each <p> element
::before Used to insert content before the content of an element p::before Insert content before the content of each <p> element
::first-letter Selects the first letter of the first line of a block level element p::first-letter Selects the first letter of all <p> elements
::first-line Selects the first line of a block level element p::first-line Selects the first line of all <p> elements
::marker Selects the marker box of a list item li::marker Selects the markers of <li> elements
::selection Selects the part of the document that has been highlighted by the user p::selection Selects part of <p> element highlighted by user
